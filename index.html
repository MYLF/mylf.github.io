Public class与class

  1. 一个Java文件中可以定义多个class，并且会生成所对应每个class的.class文件
  2. 一个Java文件中可以没有public class
  3. public class的名字必须要和Java源文件名字一致，并且每个java源文件中只能有一个
  4. 每一个class里面都可以设置main方法，都可以设置程序入口
